DROP_AND_CONNECT_INSTRUCTION_HTML=<b>拖放与连接</b>\
\
This question presents you with objects and connectors. The question asks you to create relationships between the objects by using the connectors. You create these relationships by selecting two objects and then selecting a connector.\
\
To create a relationship, take the following actions:\
<ol>\
<li>Drag two objects to an available area on the screen.\
<li>Select the objects.\
<ul>\
<li>To select the first object, click it. The color of the object will change.\
<li>To select a second object, press SHIFT and click the second object. You can select only two objects at a time.\
<li>To cancel the selection of an object, click any other object. The object that was originally selected will change back to its original color, and the other object will become selected.\
</ul>\
<li>Select the connector that you want to place between the two objects. You should now see the complete relationship on the screen.\
</ol>\
\
The connector will point from the first object you select to the second object you select. To change the direction of the connector, right-click the text of the connector, and click <b>Reverse Connection</b>.\
\
You can use two methods to remove a connector:\
<li>Right-click the text of the connector that you want to remove, and click <b>Delete</b>.\
<li>Select the text of the connector that you want to remove, and press the DELETE key.\
\
To remove from the screen all of the relationships that you have created, click the <b>Reset</b> button.\
\
补充说明：\
<li>您可能需要使用滚动条查看完整问题。\
<li>You may not need to use all of the provided answer choices.\
<li>Connectors can be used more than once.\
<li>Moving connected objects will not change the relationship between the objects.\
<li>Only objects with connectors to other objects will be scored.\
<li>The shading in the connector list is used to distinguish between items in the list.

CREATE_TREE_INSTRUCTION_HTML=<b>创建树形图</b>\
\
This question asks you to create a tree structure.\
\
The tree structure includes three levels of nodes, each marked with distinct colors and shapes:\
<li>Red triangles indicate nodes to which you cannot add items.\
<li>Blue squares indicate nodes to which you can add items.\
<li>Green circles appear next to items that you have added to nodes.\
\
You can use two methods to add an item to the tree structure:\
<li>Drag the item from the list of answer choices in the right pane of the screen to the appropriate node of the tree in the left pane of the screen.\
<li>Select the appropriate node of the tree in the left pane of the screen.\
<li>Select the appropriate item from the list of choices in the right pane of the screen.\
<li>Click the <b>Move</b> button.\
\
You can use two methods to remove an item from the tree structure:\
<li>Drag an item from the tree to the list of answer choices.\
<li>Select the item and click the <b>Remove</b> button.\
\
To remove from the tree structure all of the items that you have added, click the <b>Reset</b> button.\
\
补充说明：\
<li>You might need to use a scroll bar to view the entire text in either of the two panes.\
<li>You may not need to use all of the provided answer choices.\
<li>An answer choice can be added only to the nodes that are marked with blue squares.\
<li>An answer choice can be used more than once.\
<li>A node can have more than one choice added to it.\
<li>You might or might not need to place an answer choice at each node.\
<li>You cannot add or remove nodes that are marked with blue squares or red triangles.\
<li>The shading in the tree and in the list of choices is used to distinguish between items.\
<li>You can collapse or expand the tree by clicking the minus sign (-) or the plus sign (+). Clicking the <b>Collapse</b> button collapses the tree completely; clicking the <b>Expand</b> button expands the tree completely.
